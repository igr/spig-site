# Site

Contains just global meta-data. Defined in `src/site.json`

Contains data like:

* `site` - sites name
* `baseURL` - base URL
* `version`
* `data` - all the data collected from `/data/**/*.json` files
* `build` - various build-related information
* `dev` - various dev-related information, like all the paths
